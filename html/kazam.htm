KAZAM

This is the homepage of the Kazam project. Kazam is a visual editor and project manager for interactive game creation with Inform. It is developed with the wxWindows cross-platform GUI package, and there will be versions for Linux and Macintosh available once the program is stable.

On this page you will find the latest version and project updates.

GETTING STARTED:

Download the latest version and use Edit/Preferences to set up your Inform compiler/interpreter.

NEWS:

Oct 26, 2003: Version 0.2 released with these improvement (and more):

  • Various map view improvements:
    - Zoom
    - Improved line drawing
    - Faster updates with less flicker
    - Multiple selection with Ctrl-click or rubber band
    - Door wizard (press Shift while making a connection)
    - Calculates reasonable placement for rooms in imported code
  • Context sensitive help support (push F1 when the map view is active)
  • Find/Replace/Cut/Paste/Edit support added
  • Dropping Inform source files on the main frame works
  • Command line arguments added
  • The file dialog changes the current directory like MS Window users would expect.
  • Many small bugfixes

Oct 15, 2003: Inform source file parsing is now at least ten times faster than before and can handle very big projects.

Oct 13, 2003: new version of Kazam that is fairly stable and can handle multiple source file projects. All edit fields except those on the "Filter" page are supposed to work.
